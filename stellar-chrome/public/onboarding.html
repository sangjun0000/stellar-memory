Stellar Memory

설치 한 번이면, 모든 AI가 나를 기억합니다

🎯 지원하는 AI 사이트

🤖 ChatGPT
💬 Claude
✨ Gemini

이 3개의 AI 사이트에서 하는 대화가 자동으로 기억되고, 다음 대화에서 관련 기억이 AI에게 자동 전달됩니다.

🚀 시작하기

1

Stellar Memory 서버 설치

터미널(명령 프롬프트)에서 아래 명령을 실행하세요:

# Python이 설치되어 있어야 합니다
pip install stellar-memory[server]
2

서버 시작

설치 후 서버를 실행하세요:

stellar-memory serve-api

"Server running on http://localhost:9000" 메시지가 보이면 성공!

3

AI와 대화하기

ChatGPT, Claude, 또는 Gemini에 접속하여 평소처럼 대화하세요. Stellar Memory가 자동으로 기억합니다.

💡 사용 팁

팝업: 확장 아이콘을 클릭하면 기억 목록, 검색, 설정을 볼 수 있습니다.

자동 주입: 기본적으로 대화 시 관련 기억이 자동으로 AI에게 전달됩니다. 팝업에서 수동 모드로 전환할 수 있습니다.

사이트별 제어: 팝업에서 각 AI 사이트를 개별적으로 켜고 끌 수 있습니다.

프라이버시: 모든 데이터는 내 컴퓨터에만 저장됩니다. 외부 서버로 전송되지 않습니다.
