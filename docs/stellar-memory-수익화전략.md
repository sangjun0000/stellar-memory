## 1. 시장 분석

### 1.1 AI 메모리 시장 현황

AI 에이전트 시장은 2025~2026년 폭발적으로 성장하고 있으며, 모든 AI 에이전트의 공통된 문제점은 **기억 부재**입니다:

- **ChatGPT/Claude/Gemini**: 대화가 끝나면 기억 사라짐 (컨텍스트 윈도우 한계)
- **AI 에이전트 프레임워크** (LangChain, CrewAI, AutoGen): 기억 관리가 원시적 (단순 벡터 DB 검색)
- **기업 AI 도입**: "AI가 맥락을 기억하지 못한다"는 반복적인 불만

**핵심 인사이트**: AI 메모리는 아직 표준이 없는 **블루오션 시장**이며, Stellar Memory의 천체 구조 기반 접근은 기존 단순 벡터 DB와 차별화된 독자적 포지션을 갖고 있음.

### 1.2 경쟁 분석

| 기존 솔루션 | 방식 | 한계 | Stellar Memory 차별점 |
|------------|------|------|---------------------|
| Pinecone/Weaviate | 단순 벡터 DB | 중요도 구분 없음, 다 똑같이 저장 | 5존 기억 계층 + 기억 함수 |
| LangChain Memory | 대화 히스토리 | 컨텍스트 윈도우 초과 시 잘림 | 무제한 기억 + 자동 요약 |
| Mem0 | AI 메모리 레이어 | 단순 CRUD | 차등 망각 + 연상 기억 + 블랙홀 방지 |
| Zep | 세션 기억 | 세션 종료 시 약화 | 영구 기억 + 존 기반 자동 관리 |

**Stellar Memory만의 독점적 강점**:
1. **블랙홀 방지 알고리즘** (로그 함수 기반 - 특허 출원 가능)
2. **천체 구조 5존 모델** (직관적이고 독창적인 UX)
3. **CRDT 기반 멀티 에이전트 동기화** (다른 메모리 솔루션에 없음)
4. **감사 로그 + RBAC 보안** (엔터프라이즈 요구사항 충족)

  https://api.stellarmemory.io/v1/store \
  -d '{"content": "사용자 선호: 다크 모드"}'

## 5. 지적 재산 & 경쟁 우위 보호

### 5.1 특허 출원 가능 항목

| 항목 | 설명 | 독창성 |
|------|------|--------|
| 기억 함수 | `I(m) = w₁R + w₂F + w₃A + w₄C + w₅E` 로그 기반 블랙홀 방지 + 감정 가중치 | 높음 |
| 천체 구조 존 배치 | 5존 궤도 기반 기억 계층화 | 높음 |
| 감정-망각 연동 | 감정 강도 기반 차등 망각 속도 조절 (P7) | 높음 |
| 차등 망각 | 중요도 비례 Adaptive Decay 곡선 | 중간 |
| CRDT 기억 동기화 | LWW + Vector Clock 기반 멀티 에이전트 기억 머지 | 중간 |

### 5.2 경쟁 해자 (Moat)

1. **기술 해자**: 기억 함수 + 블랙홀 방지는 깊은 도메인 지식 필요
2. **네트워크 효과**: 멀티 에이전트 동기화 사용자가 많을수록 가치 증가
3. **전환 비용**: 기억 데이터가 축적될수록 이탈 어려움
4. **브랜드**: "천체 구조 AI 기억" = 직관적이고 기억하기 쉬운 브랜딩
5. **오픈소스 커뮤니티**: 기여자 네트워크가 곧 개발 인력

1. **코어 엔진 오픈소스** → 개발자 커뮤니티 확보
